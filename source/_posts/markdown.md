###### 简单介绍基本语法


一、标题
```
#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题
```
效果：
#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题

---

二、字体

* 加粗
``` 
**加粗文字**
```

* 斜体
```
*斜体文字*`
```

* 斜体加粗
```
***斜体加粗文字***
```

* 删除线
```
~~删除线文字~~
```

效果：

**加粗文字**
*斜体文字*
***斜体加粗文字***
~~删除线文字~~
